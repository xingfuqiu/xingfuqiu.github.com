SQL Server 2008 中文乱码解决方案

邱兴福
2012-07-11

昨天一同事用到SQL Server 2008向数据库写入汉字时，老出现乱码，在网上查了好多资料，都没有这方面的解释，不想今日和sql2000一比，竟找到原因。原来我在建数据库时用了默认的排序规则SQL_Latin1_General_CP1_CI_AS，只需要将它改为 Chinese_PRC_CI_AS就OK了。
